#### Startup
1. Clone application:
   ```shell
   git clone __REPO__
   ```
2. Go to application folder:
   ```shell
   cd __DIST__
   ```
3. Install dependencies yarn:
   ```shell
   yarn install
   ```
4. Start application in dev mode:
   ```shell
   yarn dev
   ```
5. Build in production:
   ```shell
   yarn build
   ```
6. Open website in your browser via http://localhost:3000 link.
